# Zip Knowd Website

The official GitHub Pages site for [Zip Knowd](https://zipknowd.com) — the hyperlocal app that connects neighbors by ZIP code.

## Pages

- **index.html** — Home page with hero, features, how it works, categories, map preview, and download CTA
- **about.html** — Mission, story, and values
- **contact.html** — Contact form and info
- **business.html** — Business verification and pricing plans
- **privacy.html** — Privacy policy
- **terms.html** — Terms of service

## Assets

- `css/style.css` — Shared design system and styles
- `js/main.js` — Mobile nav, scroll animations, form handling

## Brand Colors

Derived directly from the iOS app:
- Turquoise `#00D4C0`
- Blue `#4F8EF7`
- Purple `#8B5CF6`
- Coral `#FF8C5A`
- Warm background `#FAF6F0`

## Local Development

Open any HTML file directly in a browser, or serve the folder with a static file server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
